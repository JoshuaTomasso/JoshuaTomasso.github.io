Overview

FPSZombie is a multiplayer-focused Unity prototype built around cooperative or competitive zombie shooting gameplay. The most interesting part of the project is the online layer: getting players into a lobby, joining the same session, and creating a playable shared experience.

This page is set up so you can expand it further with exact systems you built, engineering tradeoffs, screenshots of code, and gameplay footage from real sessions.

What I worked on

  • Built or supported the multiplayer session flow using Unity services such as Lobby and Relay.
  • Worked in C# inside Unity to connect gameplay features to the networking layer.
  • Helped shape a 4-player experience where players could enter and play in the same shared game session.
  • Created a foundation you can keep iterating on with UI, enemy behavior, weapons, replication polish, and match flow.
