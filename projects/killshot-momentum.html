Overview

Killshot Momentum is a movement-driven first-person game built in Unreal Engine as part of a team. The project centered on creating a tight, responsive gameplay loop where momentum and movement feel were the core experience.

My contributions focused on gameplay programming in C++, working within a shared codebase to implement and support player-facing systems.

  • First-person gameplay programming in Unreal Engine using C++
  • Collaborative team development on a shared project
  • Movement and player input systems
  • Gameplay feel and responsiveness tuning
