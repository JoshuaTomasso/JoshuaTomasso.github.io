Overview

Killshot Momentum is a movement-driven first-person project created in Unreal Engine. For a portfolio, this is a good example of collaborative development and gameplay contribution inside a more performance-sensitive and input-heavy game style.

  • Supported a game where movement feel and responsiveness are key to the player experience.
  • Collaborated across a shared project scope instead of working in complete isolation.
